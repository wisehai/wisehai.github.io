反复折腾的这些年

十年前还在读书的时候，就想有个自己的网站——那样布局和内容都能自己说了算。买过服务器和域名，用 WordPress 模板搭起过一个博客，写写停停，最后总觉得内容像是寄居在别人的模板里，编辑器也用得别扭，渐渐就放下了。

后来又用 MkDocs、Hugo 各试过一遍，学会了 GitHub 的用法，也折腾出不少细节（评论功能、主题配置），但每次都卡在同一个地方：构建、部署、维护，步骤一多，就不太想更新了。

这次想明白了一件事：网站值不值得留下，跟工具复杂不复杂没关系。所以这一版故意做得很简单——没有博客系统，没有构建步骤，只留一个主页，记录正在做的事。
